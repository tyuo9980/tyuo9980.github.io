Impossibru!

Uses PCPC

Github Repo

Impossibru! is a clone of the popular The Impossible Game on XBLA. It's a crazy addictive 2D side-scroller with a soundtrack that makes you want to play more. The premise of this game is to reach the end of the level by avoiding to land on certain objects. You move at a constant speed and can only jump. Seems simple doesn't it? But it get's really tricky since some sequences trick you into jumping to your death.

The game was fairly simple to code. Collisions only consisted of rectangle to rectangle detection. I created the map by using ascii characters to represent objects in a text file.

Fun fact: I finally beat the first level on the 1057th attempt.
